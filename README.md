# gemoetry-volumen-app

# Descripcion del Proyecto

Este proyecto es una calculadora de volumen geometrica simplre construida con Python. Permite a los usuarios calcular el volumen de cuatro figuras tridimensionales: Caja, Cilindro, Cono y Esfera. La aplicacion sigue principios de programacion modular e incluye pruebas unitarias automatizadas para garantizar la precision matematica.
